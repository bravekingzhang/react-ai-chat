# react-ai-box

基于 react-native 的 AI 盒子，用于学习 react-native

- ui 组件库  react-native-elements
- 状态管理 zustand
- 路由 expo-router
- 网络请求 tanstack / react-query
- 本地缓存 async-storage

### Usage


### fix package

```bash
npx expo install --fix
```

### run

```bash
npx expo start
```
